## 1. 이론적 배경: 점탄성 역학 모델과 시간 의존적 거동

### 1-1. 점탄성(Viscoelasticity)의 정의

![점탄성의 정의와 구분](../../assets/week7/viscoelastic_concept.png)

- **개념**: 탄성(Elastic)과 점성(Viscous) 응답이 동시에 발현되는 생체 재료 고유의 복합 거동
- **순수 탄성체**: $\sigma = E \varepsilon$ — 즉각 변형, 즉각 복원 (시간 의존성 없음)
- **순수 점성체**: $\sigma = \eta \dot{\varepsilon}$ — 영구 변형, 복원력 제로 (시간 지배적)
- **점탄성체**: 탄성 복원(부분적) + 점성 영구변형이 **시간 스케일**에 따라 공존
  - 사과, 배, 치즈, 빵 반죽, 젤리, 전분 젤 등 대부분의 생물자원 해당

### 1-2. 크리프(Creep)와 응력 이완(Stress Relaxation)

![크리프와 응력 이완 곡선](../../assets/week7/creep_relaxation_curves.png)

- **크리프**: 일정 응력($\sigma_0$) 하에서 시간에 따라 변형이 점진적으로 증가
  - 적재·운송 중 과일 하단 압축 변형 예측에 활용
- **응력 이완**: 일정 변형($\varepsilon_0$) 하에서 시간에 따라 내부 응력이 점진적으로 감소
  - 충격 후 잔류 응력 평가, 포장재 설계에 활용
- **이완 시간 (Relaxation Time)**: $\tau_r = \eta / E$ — 응력이 초기값의 $1/e$ (~36.8%)로 감소하는 시간

### 1-3. 스프링-대시포트 역학 모델

| 모델 | 구조 | 강점 | 약점 |
|:---:|:---:|:---|:---|
| Maxwell | 🔩─🫙 직렬 | 응력 이완 우수 묘사 | 크리프: 직선 증가(비현실) |
| Kelvin-Voigt | 🔩═🫙 병렬 | 크리프 우수 묘사 | 응력 이완: 즉시 완전 이완(비현실) |
| Burgers | Maxwell + KV 직렬 | 크리프 + 회복 + 영구변형 포괄 | 4-파라미터 → 피팅 난이도 ↑ |

### 1-4. Maxwell 모델 수식

![Maxwell 모델 응력 이완 및 크리프 거동](../../assets/week7/maxwell_model_curves.png)

- **응력 이완**: $\sigma(t) = \sigma_0 \cdot \exp\left(-\frac{t}{\tau_r}\right), \quad \tau_r = \frac{\eta}{E}$
- **크리프**: $\varepsilon(t) = \frac{\sigma_0}{E} + \frac{\sigma_0}{\eta} \cdot t$ (직선 — 비현실적)

### 1-5. Kelvin-Voigt 모델 수식

![Kelvin-Voigt 모델 크리프 거동](../../assets/week7/kelvin_voigt_model_curves.png)

- **크리프**: $\varepsilon(t) = \frac{\sigma_0}{E}\left(1 - \exp\left(-\frac{t}{\tau_c}\right)\right), \quad \tau_c = \frac{\eta}{E}$
- ✅ 포화 곡선 형태 → 현실적 크리프 묘사
- ❌ 즉시 탄성 변형 표현 불가

### 1-6. Burgers 4-요소 모델 수식 (마스터 모델)

![Burgers 모델 크리프 변형 성분 분해](../../assets/week7/burgers_model_breakdown.png)

$$\varepsilon(t) = \underbrace{\frac{\sigma_0}{E_1}}_{\text{즉시 탄성}} + \underbrace{\frac{\sigma_0}{E_2}\left(1-e^{-t/\tau_c}\right)}_{\text{지연 탄성}} + \underbrace{\frac{\sigma_0}{\eta_1} \cdot t}_{\text{점성 유동}}$$

- $\tau_c = \eta_2 / E_2$ (지연 시간)
- **4개 파라미터**: $E_1$ (즉시 탄성), $E_2$ (지연 탄성), $\eta_1$ (점성 유동), $\eta_2$ (지연 점성)

### 📊 파이썬 스크립트 핵심 해설 (Step 1 ~ Step 3)

#### 2-1. [Step 1] Maxwell 응력 이완 시뮬레이션
- **알고리즘 코어**: $\sigma(t) = \sigma_0 \cdot e^{-t/\tau_r}$ 수식의 코드 구현
- **입력 파라미터**: 초기 응력 $\sigma_0$, 영률 $E$, 점성 계수 $\eta$
- **핵심 로직**:
  ```python
  sigma_t = sigma_0 * np.exp(-t / tau_r)
  ```
- **결과 인사이트**: $\eta$ 증가 시 이완 시간 증대 → 곡선이 우측으로 이동 → 더 오래 응력 유지
- **시각화**: 3가지 $\eta$ 값에 대한 이완 곡선 중첩 비교 + $\tau_r$ 수평선 표시

#### 2-2. 📈 [Step 2] Burgers 크리프 모델 곡선 피팅
- **알고리즘 코어**: `scipy.optimize.curve_fit` 기반 4-파라미터($E_1, E_2, \eta_1, \eta_2$) 비선형 역산
- **수식 구현**:
  ```python
  def burgers_creep(t, E1, E2, eta1, eta2):
      sigma_0 = 10.0
      instant = sigma_0 / E1
      delayed = (sigma_0 / E2) * (1 - np.exp(-t * E2 / eta2))
      viscous = (sigma_0 / eta1) * t
      return instant + delayed + viscous
  ```
- **관찰 포인트**
  - 즉시 탄성 변형(수평 점프) + 지연 탄성(곡선) + 점성 유동(직선) 3단계 분해 가시화
  - 피팅 $R^2$ 결정계수 0.99 이상 달성 여부 확인

#### 2-3. 🎛️ [Step 3] 인터랙티브 점탄성 시뮬레이터
- **UI 조작 인터페이스**
  - 모델 전환 라디오 버튼: Maxwell / Kelvin-Voigt / Burgers 선택
  - 파라미터 슬라이더: $E_1$ (50~500 kPa), $E_2$ (30~300 kPa), $\eta_1$ (500~10000 kPa·s), $\eta_2$ (100~2000 kPa·s)
  - 듀얼 플로팅: 좌측(크리프 곡선) + 우측(응력 이완 곡선) 동시 렌더링
- **관찰 포인트**
  - Maxwell → 크리프: 직선 증가 (비현실) vs Burgers → 크리프: 포화 곡선 (현실적)
  - $\eta_1$ 극대화 시 → 영구 변형 최소화 → 탄성 고체에 수렴
  - $E_1$ 극소화 시 → 즉시 탄성 점프 소실 → 점성 유체에 수렴

#### 2-4. 🚀 [Advanced] 파라미터 감도 분석 챌린지
- **목적**: $E_1$을 2배 증가, $\eta_1$을 50% 감소 후 크리프 곡선 변화 비교
- **실습 과제**:
  1. Step 2에서 피팅된 파라미터 기록
  2. $E_1 \times 2$, $\eta_1 \times 0.5$ 적용 후 크리프 곡선 재생성
  3. 원본 vs 변경본 크리프 곡선 중첩 비교 그래프 작성
- **관찰 포인트**: $E_1$ 증가 → 즉시 탄성 변형 감소, $\eta_1$ 감소 → 점성 유동 기울기 증가 → 영구 변형 증대

#### 2-5. 🚀 [Advanced] 일반화 Maxwell 모델 피팅 챌린지
- **목적**: 단일 지수 Maxwell 모델 대신 2-term 일반화 Maxwell 모델로 응력 이완 피팅 정밀도 향상
- **수식**: $\sigma(t) = \sigma_e + \sigma_1 e^{-t/\tau_1} + \sigma_2 e^{-t/\tau_2}$
- **실습 과제**:
  1. `curve_fit`으로 5-파라미터($\sigma_e, \sigma_1, \tau_1, \sigma_2, \tau_2$) 역산
  2. 단일 Maxwell($R^2$) vs 일반화 Maxwell($R^2$) 비교
- **관찰 포인트**: 2-term 모델이 다중 이완 시간 분포를 포착 → $R^2 > 0.99$ 달성 가능

## 3. 💡 심화 토론 주제 (Discussion Topics)

### 토론 1: 온도와 점탄성 파라미터의 상관관계
- **배경**: 냉장($4^\circ C$) vs 상온($25^\circ C$) 보관 시 사과의 $E$, $\eta$ 변화 양상
- **논제**: 온도 상승에 따른 이완 시간($\tau_r$) 축소 현상의 분자적 원인과, 냉장 운송이 적재 변형 저감에 미치는 정량적 효과 토론

### 토론 2: 숙성(Ripening)에 의한 점탄성 열화 메커니즘
- **배경**: 펙틴 가수분해 → 세포벽 네트워크 약화 → $E$ 급감, $\eta$ 감소
- **논제**: 점탄성 모니터링을 통한 비파괴 숙성도 판정 가능성 및 실시간 선별 연동 방안 토론

### 토론 3: 진동(Vibration) 환경과 크리프 가속
- **배경**: 트럭 운송 중 반복 진동 하중이 정적 크리프 대비 변형 가속 여부
- **논제**: 동적 피로(Dynamic Fatigue) 개념과 완충재 설계의 통합 논의. 진동 주파수별 크리프 가속 계수(Acceleration Factor) 추정 방법론

### 토론 4: Burgers 모델의 한계와 분수차 점탄성 모델
- **배경**: 4-파라미터 Burgers 모델도 실제 생물자원의 다중 이완 시간 스펙트럼을 완벽히 포착 불가
- **논제**: 분수차 미적분(Fractional Calculus) 기반 Scott-Blair 모델의 물리적 의미와, Burgers 대비 피팅 정밀도 개선 가능성

### 토론 5: 크리프 데이터 기반 최대 적재 높이 역산
- **배경**: 사과 20kg 박스를 $n$단 적재 시 최하단 과일의 24시간 누적 크리프 변형 예측
- **논제**: Burgers 파라미터 기반 허용 변형률($\varepsilon_{allow} \approx 3\%$) 초과 여부 판정 알고리즘 및 냉장 운송 시 적재 단수 증가 가능성 정량 분석
